# Mustaqbal Bot 🤖

بوت تيليجرام ذكي لمراقبة المشاريع الجديدة على منصة مستقل



## المميزات ✨

- 📱 واجهة سهلة وسلسة
- 🔔 إشعارات فورية بالمشاريع الجديدة
- 🏷️ تصفية حسب التصنيفات المفضلة
- 📊 إدارة تصنيفات متعددة
- 💾 حفظ بيانات المستخدمين

## المتطلبات

- Python 3.9+
- pip

## التثبيت

### 1. استنساخ/إنشاء المشروع
```bash
cd C:\xampp\htdocs\BotTelegram
```

### 2. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 3. إعداد متغيرات البيئة
```bash
copy .env.example .env
```

ثم عدّل ملف `.env` وأدخل:
- `TELEGRAM_TOKEN`: توكن بوتك من [@BotFather](https://t.me/botfather)
- `ADMIN_ID`: معرّف تيليجرام الخاص بك

### 4. تشغيل البوت
```bash
python bot.py
```

## الأوامر المتاحة 📋

| الأمر | الوصف |
|------|-------|
| `/start` | بدء البوت واختيار التصنيفات |
| `/categories` | تعديل التصنيفات المراقبة |
| `/stop` | إيقاف المتابعة |

## هيكل المشروع 📁

```
BotTelegram/
├── bot.py              # البوت الرئيسي
├── database.py         # إدارة قاعدة البيانات
├── scraper.py          # تجميع البيانات من مستقل
├── requirements.txt    # المتطلبات
├── .env.example        # مثال متغيرات البيئة
├── .env                # متغيرات البيئة (لا تشارك!)
└── data/
    └── bot_data.db     # قاعدة البيانات
```

## كيفية العمل 🔄

1. المستخدم يبدأ البوت بـ `/start`
2. يختار التصنيفات التي يريدها
3. البوت يراقب المشاريع كل 5 دقائق (قابل للتعديل)
4. عند ظهور مشروع جديد، يُرسل إشعار فوراً
