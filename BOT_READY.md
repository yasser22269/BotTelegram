## 🚀 تشغيل البوت الآن

### الخطوة 1: تثبيت المكتبات (إن لم تثبت بعد)
```bash
cd C:\xampp\htdocs\BotTelegram
pip install -r requirements.txt
```

### الخطوة 2: تشغيل البوت
```bash
python bot.py
```

---

## ⚙️ ما تحتاج لتعديله في .env

قبل التشغيل، عدّل هذا السطر في ملف `.env`:

```
ADMIN_ID=your_telegram_id_here
```

استبدل `your_telegram_id_here` برقم معرفك على Telegram.

**كيفية الحصول على معرفك:**
1. افتح [@userinfobot](https://t.me/userinfobot)
2. اكتب `/start`
3. انسخ الرقم (Your user ID)

---

## 🧪 اختبار البوت

بعد التشغيل:
1. افتح Telegram
2. ابحث عن `@Freelancer_Here_bot`
3. اكتب `/start`
4. اختر التصنيفات
5. بهنيء! ✅

---

## 📋 الأوامر الأساسية

| الأمر | الوصف |
|------|--------|
| `/start` | بدء البوت |
| `/categories` | تغيير التصنيفات |
| `/stop` | إيقاف المتابعة |

## ✨ المزايا الإضافية

### إضافة وصف البوت
```bash
/setdescription Freelancer_Here_bot
أنا بوت ذكي لمراقبة المشاريع الجديدة على منصة مستقل. أرسل لك المشاريع الجديدة فور نزولها!
```

### إضافة صورة ملف شخصي
1. افتح [@BotFather](https://t.me/botfather)
2. اكتب `/mypics`
3. اختر بوتك `@Freelancer_Here_bot`
4. أرسل صورة

### إضافة قائمة أوامر
```bash
/setcommands@Freelancer_Here_bot
start - بدء البوت واختيار التصنيفات
categories - تغيير التصنيفات
stop - إيقاف المتابعة
```
