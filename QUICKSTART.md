# ⚡ البدء السريع - بوت مستقل

## 🎯 في 5 دقائق فقط!

### الخطوة 1️⃣: تحميل Python
أنزّل من: https://www.python.org/downloads/

### الخطوة 2️⃣: إنشاء بوت Telegram
1. افتح [@BotFather](https://t.me/botfather)
2. اكتب `/newbot`
3. انسخ **Token** 📋

### الخطوة 3️⃣: الحصول على معرّفك
افتح [@userinfobot](https://t.me/userinfobot) واكتب `/start`

### الخطوة 4️⃣: تثبيت المشروع
```bash
cd C:\xampp\htdocs\BotTelegram
pip install -r requirements.txt
```

### الخطوة 5️⃣: الإعدادات
```bash
# أنشئ ملف .env
copy .env.example .env
```

ثم افتح `.env` وأضف:
```
TELEGRAM_TOKEN=your_token_here
ADMIN_ID=your_id_here
```

### الخطوة 6️⃣: التشغيل
```bash
python bot.py
```

### الخطوة 7️⃣: جرّب البوت
1. افتح تيليجرام
2. ابحث عن بوتك
3. اكتب `/start`
4. اختر التصنيفات
5. بهنيء! ✅

## 🆘 مشاكل شائعة

**البوت لا يرد؟**
```bash
# تأكد من:
1. التوكن صحيح في .env
2. المعرف صحيح في .env
3. الإنترنت متصل
4. البوت يعمل في Terminal
```

**خطأ في Python؟**
```bash
# حدّث pip
python -m pip install --upgrade pip

# ثم ثبّت المتطلبات
pip install -r requirements.txt
```

**قاعدة البيانات لا تعمل؟**
```bash
# احذف المجلد
rmdir /s /q data

# وأعد التشغيل
python bot.py
```
