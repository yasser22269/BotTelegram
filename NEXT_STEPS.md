## 🎨 إضافة وصف وصورة للبوت (اختياري)

### إضافة وصف البوت
1. افتح [@BotFather](https://t.me/botfather)
2. اكتب `/setdescription`
3. اختر `@Freelancer_Here_bot`
4. أكتب الوصف:
```
أنا بوت ذكي يراقب المشاريع الجديدة على منصة مستقل ويرسلها لك فوراً. 
اختر التصنيفات التي تهمك واستقبل الإشعارات من أول لحظة! 🚀
```

### إضافة صورة ملف شخصي
1. افتح [@BotFather](https://t.me/botfather)
2. اكتب `/setuserpic`
3. اختر `@Freelancer_Here_bot`
4. أرسل صورة (يفضل 640x640 بكسل)

### إضافة قائمة الأوامر
1. افتح [@BotFather](https://t.me/botfather)
2. اكتب `/setcommands`
3. اختر `@Freelancer_Here_bot`
4. أرسل:
```
start - بدء البوت واختيار التصنيفات
categories - تغيير التصنيفات المختارة
stop - إيقاف المتابعة والبوت
help - مساعدة وشرح الأوامر
```
