## 🚀 كيفية البدء

### الخيار 1️⃣: البدء السريع (5 دقائق)
```bash
1. pip install -r requirements.txt
2. copy .env.example .env
3. # عدّل .env وأضف توكنك
4. python bot.py
```

### الخيار 2️⃣: البدء المفصل (15 دقيقة)
1. اقرأ [`INSTALL.md`](INSTALL.md) كاملاً
2. اتبع كل خطوة بحذر
3. شغّل `python test_bot.py` للاختبار
4. شغّل `python bot.py`

### الخيار 3️⃣: فهم المشروع أولاً
1. اقرأ [`README.md`](README.md) - نظرة عامة
2. اقرأ [`TECHNICAL_DOCS.md`](TECHNICAL_DOCS.md) - التفاصيل
3. اقرأ [`FILE_GUIDE.py`](FILE_GUIDE.py) - شرح الملفات
4. ثم قم بالتثبيت والتشغيل
