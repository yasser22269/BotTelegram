# 🚀 بوت مستقل - Mustaqbal Telegram Bot

**بوت ذكي لمراقبة المشاريع الجديدة على منصة مستقل**

---

## ⚡ البدء السريع

### 1. التثبيت (دقيقتان)
```bash
pip install -r requirements.txt
```

### 2. الإعداد (دقيقة)
```bash
copy .env.example .env
# ثم عدّل .env وأضف توكنك
```

### 3. التشغيل (ثانية)
```bash
python bot.py
```

### 4. الاستخدام
- افتح Telegram وابحث عن بوتك
- اكتب `/start`
- اختر التصنيفات
- استقبل المشاريع! 🎉
